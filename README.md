# Atividade: Representações de Grafos

Repositório contendo implementações de grafos para a disciplina "Resolução de Problemas de Grafos".

Representações incluídas:
- `adjacency_list.py` — Lista de Adjacência
- `adjacency_matrice.py` — Matriz de Adjacência
- `adjacency_edges.py` — Lista de Arestas

>Observação: o estilo de código segue o mesmo padrão nas três implementações.

Integrantes da dupla:
- Guilherme Lopes Bertacini - 2011593
- Paulo Henrique Fusco - 2011245

Requisitos
- Python 3.8+ (não há dependências externas)

Como executar

Abra um terminal na pasta `graph-representation-activity` e rode um dos scripts para ver a demonstração de cada representação.

Executar a demonstração da Lista de Adjacência:

```bash
python adjacency_list.py
```

Executar a demonstração da Matriz de Adjacência:

```bash
python adjacency_matrice.py
```

Executar a demonstração da Lista de Arestas:

```bash
python adjacency_edges.py
```
